📱 AttSys

Internship Project @ Expoceylon PLC 🌐 Live System → attsys.expoceylonplc.com
🏆

⚡ Built Within 1 Month of Joining

Developed from scratch — fully created by me as my first major internship project. Complete QR attendance system with real-time validation, reporting, and special power-outage mode.

100% Solo Development

📋 Project Overview

QR Code-based Attendance Tracking System for branch employees of Expoceylon PLC. Employees scan unique QR codes to mark attendance with real-time validation and automated reporting for HR management.

👨‍💻 My Role (Solo Developer)

  • Full Stack Developer - Entire system built by me
  • QR Code Generator & Scanner Integration
  • Complete Database Architecture & Design
  • Real-time Attendance Validation Logic
  • Report Generation Module (PDF/Excel)
  • UI/UX Design & Frontend Development
  • Server Deployment & Configuration

⚙️ Key Features

  • Unique QR code per employee
  • Real-time attendance marking with timestamp
  • Device charging requirement validation
  • Power outage mode for branches
  • Automated monthly attendance reports
  • Admin dashboard & employee management

🔋 Live System Features: Employees must have device plugged into charging to mark attendance (ensures accountability). Power Outage Mode available for branches with active outages — employees can log in without charging requirement during such events.

✨ Note: This entire system was conceptualized, designed, developed, tested, and deployed by me within 1 month of joining Expoceylon PLC as an IT Intern. No external development team involved — 100% self-built solution now actively used by the company.

🛠️ Technologies Used

PHP 8.x MySQL JavaScript (Vanilla) QR Code Library (phpqrcode) Bootstrap 5 AJAX HTML5/CSS3 cURL Apache/Nginx
