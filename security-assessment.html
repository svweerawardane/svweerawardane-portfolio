Security Vulnerability Assessment

Academic security assessment project conducted on the IT infrastructure of UTE CAT Pvt Ltd to identify vulnerabilities and recommend improvements.

Project Overview

This project was conducted as part of an academic requirement for the Computer Security module. The objective was to analyze the existing IT systems of UTE CAT Pvt Ltd, identify potential security vulnerabilities, and propose mitigation strategies based on best security practices.

My Role

  • Security Analyst (Academic Project)
  • Responsible for vulnerability identification and documentation
  • Prepared structured security assessment reports

Assessment Scope

  • Basic network security review
  • System and configuration weaknesses
  • Access control and user management review
  • Risk evaluation and severity classification

Outcomes & Learnings

  • Identified security gaps and potential threats
  • Recommended practical security improvements
  • Improved understanding of real-world security risks
  • Hands-on experience in security reporting
