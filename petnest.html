PetNest

PetNest is a platform for pet owners, veterinarians, groomers, and pet sitters. Manage pets, appointments, and services efficiently in one place.

Project Overview

This project was developed as part of an assignment for the (keep module blank). The platform helps pet owners manage pet records, schedule vet visits, and find grooming or sitting services easily.

  • User-friendly dashboard for each role
