🐾 PetNest

PetNest is a platform for pet owners, veterinarians, groomers, and pet sitters. Manage pets, appointments, and services efficiently in one place.

📋 Project Overview

This project was developed as an individual project for the Web Application Development (WAD) module at SLIIT City University. PetNest serves as a comprehensive platform connecting pet owners with veterinary professionals, groomers, and pet sitters.

👨‍💻 My Role: Solo Full Stack Developer — responsible for frontend design, backend logic, database architecture, and deployment.

✨ Features

  • Pet management (add, update, view pets with medical history)
  • Appointment scheduling with veterinarians
  • Connect with groomers and pet sitters
  • User-friendly dashboard for each role (Pet Owner, Vet, Groomer, Sitter)
  • Service booking & review system
  • Secure login & registration system

🛠️ Technologies Used

HTML5 CSS3 JavaScript PHP MySQL Bootstrap AJAX
