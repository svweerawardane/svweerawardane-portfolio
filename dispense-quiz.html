📱 DispenseApp & QuizApp

Java-based desktop applications developed during a supervised lab practical examination to demonstrate GUI design and logical problem-solving skills.

📋 Project Overview

DispenseApp and QuizApp were developed as part of a lab practical examination for the Data Structures & Algorithms (DSA) module at SLIIT City University. The objective was to design functional Java desktop applications within a limited time frame, following given requirements and constraints while demonstrating proficiency in Java Swing and algorithmic thinking.

👨‍💻 My Role: Solo Developer — responsible for designing both applications from scratch, implementing GUI using Java Swing, handling user inputs, validations, and ensuring correct program flow under strict exam time constraints.

⏱️ Exam Conditions: Both applications were developed within a supervised lab session with time limitations, demonstrating ability to work efficiently under pressure while maintaining code quality and functionality.

💊 DispenseApp

A medication dispensing simulation application that tracks inventory, processes dispense requests, and manages stock levels.

  • Medicine inventory management
  • Dispense request processing
  • Stock level validation
  • Transaction history logging

❓ QuizApp

Interactive quiz application with multiple-choice questions, score tracking, and result evaluation.

  • Multiple-choice question interface
  • Real-time score calculation
  • Question navigation system
  • Final result display with feedback

✨ Key Features (Both Applications)

  • Interactive graphical user interface using Java Swing
  • Logical processing based on user actions and inputs
  • Comprehensive error handling and input validation
  • Efficient execution optimized for exam time limits
  • Clean and intuitive user experience
